# Data-Science-Internship
This repository contains my Data Science Internship projects and tasks completed using Python and Google Colab. It includes exploratory data analysis (EDA), data visualization, statistical analysis, and machine learning exercises. The repository showcases practical applications of data science concepts and tools through hands-on projects.
 
                            Task 1: Exploring and Visualizing the Iris Dataset

Work Completed
* Imported required Python libraries (Pandas, NumPy, Matplotlib, and Seaborn).
* Loaded the Iris dataset into Google Colab.
* Explored the dataset using `.shape`, `.columns`, `.head()`, `.info()`, and `.describe()`.
* Checked for missing values and verified data quality.
* Created scatter plots to analyze relationships between features.
* Generated histograms to study feature distributions.
* Created box plots to identify outliers and compare feature values across species.

* Interpreted the visualizations and summarized key findings.

                                     Task 3: Customer Churn Prediction


 Work Completed

* Loaded the Churn Modelling dataset into Google Colab.
* Removed unnecessary columns (RowNumber, CustomerId, and Surname).
* Verified that the dataset contained no missing values.
* Visualized customer demographics, age distribution, gender, geography, and churn patterns.
* Encoded categorical variables using Label Encoding and One-Hot Encoding.
* Split the dataset into training and testing sets.
* Trained a Random Forest Classifier.
* Evaluated model performance using Accuracy Score, Confusion Matrix, and Classification Report.
* Analyzed feature importance to determine the most influential variables affecting customer churn.
* Interpreted the model results and business insights.
